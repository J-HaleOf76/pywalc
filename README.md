# pwy
Change [pywal](https://github.com/dylanaraps/pywal) colors online

### Installation
```
git clone --recursive https://github.com/png261/pwy
cd pwy
chmod +x && ./install.sh
```

### Dependencies
- pywal
- feh
- uvicorn 
- xclip
- qrencode
